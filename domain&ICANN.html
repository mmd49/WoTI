Domain and ICANN

Domain Name

A name associated with an IP address a device uses to locate other devices around the world.

You can think a domain name is a sort of like an address while your IP address is like your zip code

For example, a domain name could be www.something.com that has an IP address of 123.55.234.15

We use domain names to make it easier to search and find various websites on the internet without having to remember long, complicated numbers

DNS

Domain Name Sever

A server dedicated to the storage of specialized domains names used.

These could include domains that use specific phrases such as .com, .org, and .net.

The is allows to have to rely on a single domain to manage all other domains names around the world.

ICANN

Internet Corporation for Assigned Names and Numbers

A non-profit corporation responsible for the maintenance of namespaces and numerical spaces on the internet

These people work with Domain Name Server and make sure that every device is assigned a proper and unique address throughout the world
