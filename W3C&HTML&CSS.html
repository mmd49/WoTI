W3C, HTML, & CSS

W3C

World Wide Web Consortium

An international community of people who continually work to develop and improve new web standards.

They are the people who are responsible for making sure why the internet works the way it is.

- Without them, the internet may not have been able to scale upwards with its growing demand continually.

HTML

Hypertext Markup Language

The language used to tell a web browser how to make a web page look.

This code allows for some basic stylization of web pages and can insert various media elements such as images, videos, and music.

CSS

Cascading Style Sheets

A language used with HTML allows further stylizations of web pages and can create more dynamic looks to every webpage on a site.

Without CSS, a developer would have to add style code onto every page of their website manually.

\

- With CSS, a developer can rapidly change the look of all the sites and even chose multiple styles for them.
